
What will be the output of the program?

	double f = 12.0;
	switch( f )
	{
		case 12: System.out.println("Twelve");
		default: System.out.println("Default");
	}

A.	 Zero 	B.  Twelve	C.  Default	D. Compilation error

            Solution :

The switch-expression must evaluate to a type: byte, short,
char, int, enum, or String

            Ans : D
